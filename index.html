Propiedades de CSS

Propiedades de CSS

Propiedad Valor Inicial Descripción
color inherit Define el color del texto del elemento.
font-family serif Especifica la fuente del texto del elemento.
font-size medium Establece el tamaño de la fuente del texto.
font-weight normal Determina el grosor del texto, como normal, bold, o valores numéricos (100, 400, 700).
font-style normal Define el estilo del texto, como normal, italic, o oblique.
font-variant normal Cambia el estilo de la fuente para variantes tipográficas como small-caps.
font normal normal medium serif Agrupa propiedades de fuente en una sola declaración (font-style, font-variant, font-weight, font-size, line-height, font-family).
text-align left Alinea el texto dentro de su contenedor (left, right, center, justify).
line-height normal Establece la altura de la línea, afectando el espaciado vertical entre líneas de texto.
text-decoration none Añade decoraciones al texto, como underline, overline, line-through, o none.
text-transform none Cambia la capitalización del texto (uppercase, lowercase, capitalize, none).
vertical-align baseline Alinea el texto verticalmente en relación con su línea base (baseline, top, middle, bottom).
text-indent 0 Define la cantidad de espacio para la indentación de la primera línea de un párrafo.
letter-spacing normal Ajusta el espaciado entre caracteres (normal, 0.1em, etc.).
word-spacing normal Modifica el espaciado entre palabras (normal, 0.2em, etc.).
white-space normal Controla cómo se maneja el espacio en blanco dentro del elemento (normal, nowrap, pre, etc.).

Decoración

Característica Propiedades Descripción
Tamaño, Color y Decoración font-size, color, text-decoration - `font-size`: Establece el tamaño de la fuente del texto.
- `color`: Define el color del texto.
- `text-decoration`: Añade decoraciones al texto como `underline`, `overline`, `line-through`.
Decoración Personalizada text-transform, letter-spacing, word-spacing, text-shadow - `text-transform`: Cambia la capitalización del texto (`uppercase`, `lowercase`, `capitalize`).
- `letter-spacing`: Ajusta el espaciado entre caracteres.
- `word-spacing`: Modifica el espaciado entre palabras.
- `text-shadow`: Añade sombra al texto para un efecto personalizado.

Anchura y Altura

Característica Propiedades Descripción
Anchura y Altura width, height - `width`: Establece el ancho de un elemento.
- `height`: Define la altura de un elemento.
Eliminar el Borde border, border-width - `border`: Elimina el borde de un elemento cuando se establece en `none`.
- `border-width`: Ajusta el grosor del borde. Para eliminarlo, se puede establecer en `0`.
